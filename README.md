# Flipper Menu Tree Generator

Generates menu tree image from markdown code

I hack together small PoC. It takes MD file, uses parser library from @zhovnerâ€™s example (https://github.com/gera2ld/markmap-lib 2) (so, NO YAML), translate it to dot syntax (Graphviz) and produces following image:

It is not interactive, on the other hand, generation of images could be part of CI pipeline.
